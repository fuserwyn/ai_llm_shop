# ai_llm_shop

## Доступные команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать список команд
- `/set_datetime [дата] [время]` - Установить дату и время

### Использование /set_datetime

Команда для установки даты и времени:

```
/set_datetime [дата] [время]
```

Пример использования:
```
/set_datetime 2024-12-25 15:30:00
```

Формат даты: `ГГГГ-ММ-ДД`
Формат времени: `ЧЧ:ММ:СС`
