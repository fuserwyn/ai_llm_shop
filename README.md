# ai_llm_shop

Telegram бот для работы с AI/LLM моделями через OpenRouter API.

## Возможности

- 🤖 Общение с AI ассистентом Дикси через OpenRouter
- 🕒 Показ текущего времени
- 📱 Удобное меню с кнопками
- ⚡ Быстрые ответы от различных LLM моделей

## Настройка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне проекта:
```env
BOT_TOKEN=ваш_telegram_bot_token
OPENROUTER_API_KEY=ваш_openrouter_api_ключ
ADMIN_IDS=123456789,987654321  # ID администраторов через запятую
```

3. Получите API ключ OpenRouter:
   - Зарегистрируйтесь на [OpenRouter](https://openrouter.ai)
   - Перейдите в настройки API Keys
   - Создайте новый ключ и скопируйте его

4. Запустите бота:
```bash
python main.py
```

## Использование

- `/start` - Начать работу с ботом
- `/help` - Показать список команд
- `/time` - Показать текущее время
- `/dixi` - Начать общение с Дикси (AI ассистент)
- `/menu` - Показать меню с кнопками

После команды `/dixi` просто отправляйте любые сообщения - они будут переданы AI ассистенту.

## Конфигурация OpenRouter

В файле `config.py` можно настроить:
- `OPENROUTER_MODEL` - модель по умолчанию (например, "openai/gpt-3.5-turbo")
- `OPENROUTER_BASE_URL` - базовый URL API
